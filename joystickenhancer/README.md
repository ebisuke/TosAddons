# joystickenhancer
ワールドマップ・スキル画面を開くと自動でUIモードになります。いくつかの項目にリングコマンドを追加します

# 使用方法
ワールドマップ・スキル画面を開くと自動でUIモードになります。  いくつかの項目にリングコマンドを追加します
設定は特にありません。
